Projeto sobre o jogo TCG Card Pokemon.
É um banco de dados em PostgreSQL contendo os seguintes dados:
Tabelas 
  tbl_collections
    Nome da coleção
    Data de lançamento
    Total de cartas da coleção
  tbl_types
    Elemento de ataque
  tbl_stages
    Estágios de evolução
  tbl_cards
    hp
    pokemonName
    info
    attack
    damage
    weak
    resis,
    retreat
    cardNumberInCollection
    collection_id

Os dados utilizados no preenchimento desse banco de dados foi gerado no Copilot baseando toda a estrutura criada para esse banco.
